# Personal website
This repository hosts the source code for my personal website, written in plain HTML, CSS and Javascript, with the help of [11ty/eleventy](https://www.11ty.dev/) and [Nunjucks](https://mozilla.github.io/nunjucks/) to to be able to factorize HTML into reusable pieces.

## How to run locally
To run the website locally, you need to have [Node.js](https://nodejs.org/en) installed on your machine. Once you have Node.js installed, follow these steps:

1. Clone this repository to your local machine.
2. Navigate to the root directory of the repository in your terminal.
3. Run the following command to install the required dependencies:
   ```
   npm install
   ```
4. Run the following command to start the development server:
   ```
   npm run serve
   ```
5. Open your browser and navigate to `http://localhost:8080` to view the website.
